Auralis / Private talent intake

Does this candidate journey feel right?

Six senior contractor roles share one secure flow, with role-specific questions and assessment content.

01

Instagram role post

Each campaign links to an unlisted role URL. There is no public careers directory.

02

Short application

Name, email, country, time zone, portfolio links, availability, and private CV upload.

03

Private invitation

A single-use assessment link is emailed after the application passes basic validation.

04

Role-specific MCQ

18 randomized questions, 20 minutes, no written answers, saved once and submitted once.

05

Card verification

€2.99 authorization is cancelled immediately. It never changes score, rank, or review order.

06

Completion

Candidate receives confirmation. A secure summary and temporary CV link go to hello@auralis.studio.

Privacy: assessment URLs expire, CV files stay private, and recruiter links are time-limited.
Fairness: all completed applications are reviewed; card status is operational metadata only.
